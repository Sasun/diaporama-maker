Diaporama Maker
===============
An editor to make image slideshows performed with [`diaporama`](http://github.com/gre/diaporama) â€“ including KenBurns effect and GLSL Transitions.

Installation
------------

```
npm install -g diaporama-maker
```

Requirements
-----------

- `imagemagick` installed on your computer. You can check that `convert` command is working.
- A Web Browser with WebGL support.

Running
-------

```
diaporama directory_with_images
```
